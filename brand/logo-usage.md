# Portfolio Logo Usage Guidelines

*May 2026 · paired with `Portfolio Brand Guide.md`*

---

## What Portfolio's mark is

The Portfolio wordmark is the only mark. There is no icon, monogram, or symbol — the name set in Momo Trust Display Medium is the brand.

For most contexts, use one of the wordmark SVGs in this folder. The favicon's "P" and GitHub avatar's "P" are reductions of the wordmark to a single glyph, not separate icons.

---

## Permitted uses

Partner organizations may use the Portfolio wordmark to indicate Portfolio is a partner, funder, supporter, or affiliate. Press and journalists may use the wordmark in articles about Portfolio. Project teams within Portfolio may reference the studio of origin using the wordmark on their own surfaces.

---

## Not permitted

Don't modify the wordmark — no color changes outside the two specified, no effects, no recombinations, no symbol additions, no shadows, no outlines.

Don't substitute typefaces. The wordmark must be set in Momo Trust Display Medium. Fallbacks (Nunito, system rounded sans) are acceptable only when the brand font genuinely can't be loaded.

Don't create lockups. The wordmark stands alone, never combined with taglines, secondary logos, or other graphics in a single composed mark.

Don't use the wordmark to imply endorsement Portfolio hasn't given.

---

## Clear space

Maintain at least 1× the wordmark's cap height of clear space on all sides. Cap height is roughly 70% of the wordmark's set font size — so at 22px wordmark, clear space is ~15px on all sides.

---

## Minimum size

The wordmark must render at 16px or larger on screen, or 14pt or larger in print. Below those sizes, use the favicon or GitHub avatar P-mark instead.

---

## Color

Two versions only — light gray (`#E6E6E6`) on dark grounds, ink (`#0A0A0A`) on light grounds. Never colored, gradient, outlined, or shadowed.

---

## Production exports

The shipped SVGs use Momo Trust Display via `@import`. The font loads in most modern browsers and design tools that support web fonts.

For ironclad production use — printed materials, partner deliverables, exported PNGs at specific sizes — open the SVG in a design tool (Illustrator, Figma, Sketch, Affinity Designer), confirm Momo Trust Display is installed locally, then convert the wordmark text to outlined paths before exporting. Outlined paths are font-independent and render identically everywhere.
